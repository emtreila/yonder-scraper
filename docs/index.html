Yonder Job Scraper

Automated scraper for job listings at tss-yonder.com/job.

Architecture

  • index.js — Main scraper entry point
  • company.js — Company validation via ANAF
  • solr.js — Solr storage layer
  • src/anaf.js — ANAF API integration

Data Flow

  1. Fetch HTML from tss-yonder.com/job
  2. Parse job listings with cheerio
  3. Validate company via ANAF (CUI 4906881)
  4. Upsert jobs/company to Solr
  5. Save jobs.json locally
