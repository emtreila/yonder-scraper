# Yonder Job Scraper - Configuration Files

This document describes the configuration files used by the Yonder Job Scraper.

## Files Structure

- `package.json` - Node.js project configuration and dependencies
- `.env` - Environment variables (SOLR_AUTH, SOLR_BASE)
- `.github/workflows/scrape.yml` - Daily scrape workflow
- `.github/workflows/test.yml` - Test workflow
- `.github/workflows/deploy.yml` - Optional deploy workflow
